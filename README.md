## Summary

- Scan handwritten gratitude lists via camera or photo library
- Claude AI vision API for accurate handwriting recognition
- Editable text field to correct OCR output
- Save entries with a date picker (so you can backdate)
- Saved entries screen with swipe to delete and tap to edit
- Share via iMessage or any iOS share sheet
- Olive + Gill Sans theme throughout
- App lives on my phone (can be downloaded via XCode)
